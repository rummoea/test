# Тестовое задание

Для собеседования на позицию программист C++ в Playrix. Апрель 2016

## Firework

Реализация задачи "Фейерверк с задаваемым уровнем вложенности".
Содержит файлы с исходным кодом, скомпилированное приложение и необходимые библиотеки и файлы ресурсов.
Параметры настраиваются в файле Firework\bin\input.txt, где:
* Level = "Уровень вложенности"
* Count = "Количество частиц"

Запускать Firework\bin\Firework.exe

## Bubbles

Реализация задачи "Игра пузыри".
Содержит файлы с исходным кодом, скомпилированное приложение и необходимые библиотеки и файлы ресурсов.
Параметры настраиваются в файле Bubbles\bin\input.txt, где:
* CountPoints = "Количество точек"
* BornTime = "Время увеличения круга"
* LifeTime = "Время жизни круга"

Запускать Bubble\bin\Bubbles.exe

Реализация с альтернативным оформлением Bubble\bin-lemonade\Bubbles.exe
